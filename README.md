# jdbc-database-application-java
Java JDBC application to perform CRUD operations on MySQL database with Restaurant and MenuItem tables.

Features
Insert records into Restaurant and MenuItem
Select MenuItems with price <= 100
Retrieve MenuItems from a specific restaurant ("Cafe Java")
Update MenuItem prices
Delete MenuItems based on name condition
Display results using ResultSet in tabular format

Database Schema
Restaurant(Id, Name, Address)
MenuItem(Id, Name, Price, ResId)
